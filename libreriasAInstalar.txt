Librerias a instalar en node:

Por proyecto:
npm i util // me sirve para concertir una funcion en async
npm i mysql //  me sirve para conectarme con una abse de datos mysql 
npm i dotenv // me sirve para crear archivos ocultos .env
npm i nodemailer // me sirve para conectarme con nodemailer 
npm i multer // manejador de archivos para JS
npm i uuid // genera ids unicos de 45 caracteres
npm i sha1 // encripta contras
npm i express-session // instala las sessions
npm i @hapi/joi // me sirve apra las validaciones
